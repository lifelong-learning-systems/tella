About tella
===========
tella stands for Training and Evaluating Lifelong Learning Agents.
It provides a standard API and tools for performing continual learning experiments.

tella has three primary components:
 * :ref:`Curriculums of tasks for testing continual RL agents <ABOUT:Continual RL curriculums>`.
 * :ref:`An API for user-defined agents <ABOUT:RL agent API>`.
 * :ref:`A command line interface for running RL experiments <ABOUT:Running tella>`.

Additionally, these docs contain:
 * :ref:`Installation instructions <INSTALL:Installing tella>`.
 * :ref:`Example agents <EXAMPLES:Example agents>`.
 * :ref:`Contributor's guidelines <CONTRIBUTING:Contributing>`.
